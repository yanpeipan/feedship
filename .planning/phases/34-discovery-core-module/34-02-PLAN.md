  <name>Task 1: Implement HTML link parser (parser.py)</name>
  <files>src/discovery/parser.py</files>
  <action>
Implement src/discovery/parser.py with:

1. parse_link_elements(html: str, page_url: str) -> list[DiscoveredFeed]:
   - Parse HTML string using BeautifulSoup
   - Find all <link> tags in <head> section
   - For each link with rel="alternate" (case-insensitive) and type containing rss/atom/rdf:
     - Extract href attribute
     - Resolve relative URLs using urllib.parse.urljoin(page_url, href)
     - If <base href> exists in <head>, use it for URL resolution override
     - Determine feed_type from type attribute ('rss', 'atom', 'rdf')
     - Extract title attribute if present
   - Return list of DiscoveredFeed objects with source='autodiscovery'

2. resolve_url(page_url: str, href: str, base_href: str | None = None) -> str:
   - If base_href is provided, use it as the base for urljoin instead of page_url
   - Otherwise use page_url as base
   - Return absolute URL

3. extract_feed_type(content_type: str) -> str | None:
   - Return 'rss' if 'rss' in content_type.lower()
   - Return 'atom' if 'atom' in content_type.lower()
   - Return 'rdf' if 'rdf' in content_type.lower()
   - Return None otherwise

Key implementation notes:
- Use BeautifulSoup with 'lxml' parser (already installed)
- Normalize rel and type to lowercase before comparison (DISC-01)
- Handle missing href gracefully (skip such links)
- BeautifulSoup: soup.find('head').find_all('link') to find link tags
- Check for <base href=""> before parsing link elements

Implement src/discovery/fetcher.py with:

1. validate_feed(url: str) -> tuple[bool, str | None]:
   - Send HEAD request to url using httpx
   - Return (True, feed_type) if HTTP 200 and Content-Type contains rss/atom/rdf
   - Return (False, None) if validation fails (404, wrong content-type, etc.)
   - Use BROWSER_HEADERS from rss_provider for consistent behavior
   - Use 10 second timeout, follow redirects

2. is_bozo_feed(url: str) -> tuple[bool, str | None]:
   - Fetch feed URL using httpx (GET, not HEAD - need content to parse)
   - Parse with feedparser.parse(content)
   - Return (True, error_message) if feed.bozo == 1
   - Return (False, None) if feed is well-formed
   - Run feedparser.parse in try/except - treat exceptions as bozo
   - Use 15 second timeout for full fetch

3. FEED_TYPE_MAP for mapping Content-Type to feed_type strings:

   ```python
   }
   ```
