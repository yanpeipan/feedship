1. async def discover_feeds(url: str) -> list[DiscoveredFeed]:
   Main entry point for feed discovery.

   Algorithm:
   a. Normalize input URL (ensure has scheme, add https:// if missing)
   b. Fetch page HTML using httpx.AsyncClient with BROWSER_HEADERS
   c. Parse HTML for autodiscovery link elements via parse_link_elements()
   d. If autodiscovery finds feeds:
      - Validate each via validate_feed()
      - Filter out bozo feeds via is_bozo_feed()
      - Return validated list
   e. If autodiscovery finds nothing (empty list):
      - Probe well-known paths from WELL_KNOWN_PATHS
      - For each path: validate_feed() first (skip full bozo check for speed)
      - Collect valid paths into DiscoveredFeed objects
      - Return validated list
   f. Return empty list if no feeds found

2. Helper: normalize_url(url: str) -> str:
   - If URL has no scheme, prepend 'https://'
   - If URL has only scheme://, prepend 'https://'
   - Strip trailing whitespace
   - Return normalized URL

3. Helper: probe_well_known_paths(page_url: str) -> list[str]:
   - Take base URL (remove path, keep scheme+host)
   - For each path in WELL_KNOWN_PATHS, construct full URL
   - Return list of candidate feed URLs

4. Helper: async validate_and_wrap(url: str, page_url: str, source: str) -> DiscoveredFeed | None:
   - Call validate_feed(url) - if valid, create DiscoveredFeed
   - Return DiscoveredFeed or None

5. Public exports at module level:
   ```python
   from src.discovery.models import DiscoveredFeed
   from src.discovery.parser import parse_link_elements
   from src.discovery.fetcher import validate_feed, is_bozo_feed
   from src.discovery.common_paths import WELL_KNOWN_PATHS
   ```

Key implementation notes:
- Use httpx.AsyncClient for async HTTP fetching
- Use asyncio.to_thread() for validate_feed() calls to avoid blocking
- Use asyncio.gather() for concurrent validation of multiple feeds
- Set reasonable timeouts: 10s for page fetch, 10s for HEAD requests
- Log at debug level for discovery attempts and results
- Import and reuse BROWSER_HEADERS from src.providers.rss_provider
- Store discovered title from parse_link_elements in DiscoveredFeed.title
