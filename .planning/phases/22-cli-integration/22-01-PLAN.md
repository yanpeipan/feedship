3. `fetch` command has `--concurrency` option with `default=10` and `type=click.IntRange(1, 100)`
4. When `--all` is triggered, calls `uvloop.run(fetch_all_async(concurrency=concurrency))`
5. Output displays `total_new`, `success_count`, `error_count` from the async result
6. On errors, displays error list at the end

**action:**
1. Add imports at top of `fetch` function block (line 178):
   ```python
   import uvloop
   from src.application.fetch import fetch_all_async
   ```

2. Add `--concurrency` option to `@cli.command("fetch")` decorator (line 163-166):
   ```python
   @cli.command("fetch")
   @click.option("--all", "do_fetch_all", is_flag=True, help="Fetch all feeds")
   @click.option("--concurrency", default=10, type=click.IntRange(1, 100), help="Max concurrent fetches (default: 10)")
   @click.argument("urls", nargs=-1, required=False)
   @click.pass_context
   ```

3. Update `do_fetch_all` handling (lines 208-265) to replace sequential loop with:
   ```python
   if do_fetch_all:
       try:
           feeds = list_feeds()
           if not feeds:
               click.secho("No feeds subscribed. Use 'feed add <url>' to add one.", fg="yellow")
               return

           result = uvloop.run(fetch_all_async(concurrency=concurrency))

           if result["total_new"] > 0:
               click.secho(f"  + {result['total_new']} new articles", fg="green")
           else:
               click.secho("  No new articles", fg="blue")

           if result["error_count"] > 0:
               click.secho(f"  {result['error_count']} errors occurred:", fg="yellow")
               for err in result["errors"]:
                   click.secho(f"    - {err}", fg="red")
               click.secho(
                   f"Fetched {result['total_new']} articles from {result['success_count']} feeds, "
                   f"{result['error_count']} errors",
                   fg="yellow",
               )
           else:
               click.secho(
                   f"Fetched {result['total_new']} articles from {result['success_count']} feeds",
                   fg="green",
               )
       except Exception as e:
           click.secho(f"Error: Failed to fetch feeds: {e}", err=True, fg="red")
           logger.exception("Failed to fetch feeds")
           sys.exit(1)
       return
   ```

4. Remove the now-unused imports `fetch_one` from the `fetch_all` block (line 211) since we now use `fetch_all_async`.
