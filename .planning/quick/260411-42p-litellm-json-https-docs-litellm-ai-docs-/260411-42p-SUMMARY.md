# Quick Task 260411-42p: litellm JSON Mode for NER Chain

**Completed:** 2026-04-10

**Status:** Verified ✓

## Changes

### `src/llm/chains.py`

1. **AsyncLLMWrapper** (`__init__`): 新增 `response_format: dict | None` 参数，存储为 `self._response_format`
2. **_ainvoke_raw**: 当 `self._response_format` 存在时，将其注入 `extra_body["response_format"]`，透传给 litellm
3. **_llm_wrapper_cache**: 缓存 key 从 `int` 改为 `tuple[int, frozenset]`，支持按 `response_format` 组合缓存
4. **_get_llm_wrapper**: 新增 `response_format` 参数，传递给 `AsyncLLMWrapper` 构造
5. **NER_PROMPT**: 移除冗余 JSON 说明（`"IMPORTANT: return ONLY valid JSON..."` 等），JSON mode 由 litellm 保证
6. **get_ner_chain()**: 传入 `response_format={"type": "json_object"}`，启用 litellm 原生 JSON mode

### `src/application/report/ner.py`

- `_extract_json` 函数已在上一次用户任务中删除（不在本 commit 范围内）
