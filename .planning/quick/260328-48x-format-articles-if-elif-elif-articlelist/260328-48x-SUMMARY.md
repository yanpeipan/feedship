# Phase quick-260328-48x Plan 01: Refactor format_articles Summary

## One-liner

Remove mode branching from format_articles by using unified score field interface from rank_*_results.

## Objective

Refactor format_articles to remove if/elif/elif mode branching. Since rank_*_results now adds score field consistently, CLI should call rank_*_results before format_articles. format_articles then just formats items with score field without needing mode.
