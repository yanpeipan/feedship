### Task Summary
在 `~/.local/share/feedship/templates/` 下创建 `ai-daily.md` 模板文件，解析用户提供的示例结构。

### Data Model Understanding

**Available variables from `cluster_articles_for_report()` return dict:**
- `layers`: list of `{name, topics[]}` — 按AI五层蛋糕分类的文章
  - `name`: 层名 (AI应用, AI模型, AI基础设施, 芯片, 能源)
  - `topics`: list of `{title, headline, layer, signals, sources[]}`
    - `headline`: 该话题得分最高文章的翻译标题（用作section小标题）
    - `sources`: list of article dicts — `{id, title, link, summary, quality_score, ...}`
- `date_range`: `{"since": "YYYY-MM-DD", "until": "YYYY-MM-DD"}`

### Template Structure (from user example)

```
AI 日报 — {{ date_range.since }}
A. AI五层蛋糕
1. AI应用
**headline1**
[title1](link1)
[title2](link2)
...
2. AI模型
**headline1**
[title1](link1)
...
3. AI基础设施
（本期暂无相关来源）
4. 芯片
（本期暂无相关来源）
5. 能源
（本期暂无相关来源）
```

### Key Design Decisions
- Layer index uses Arabic numerals: 1-5 (not Chinese characters)
- 每个layer下，topic的headline作为加粗小标题
- Article列表：[title](link) 格式
- Empty layer: `（本期暂无相关来源）`
- No "主编导读" section (pre-pended by CLI/editor outside template)
