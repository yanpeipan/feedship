## Task

把 `deduplicate_articles`（三重去重）接入 Report 生成流程，在 `list_articles()` 取到文章后、进入 `_entity_report_async()` 之前调用。

## Required Changes

### 1. `src/storage/sqlite/articles.py` — SELECT 增加 content_hash 和 minhash_signature

文件: `src/storage/sqlite/articles.py:510`

在 SELECT 语句中增加 `a.content_hash, a.minhash_signature` 两列。

### 2. `src/application/articles.py` — ArticleListItem 增加两个字段

文件: `src/application/articles.py:54`

```python
content_hash: str | None = None
minhash_signature: bytes | None = None
```

在 `_compute_article_item` 返回时设置这两个字段（从 row 字典取值）。

### 3. `src/application/report/report_generation.py` — 调用 deduplicate_articles

文件: `src/application/report/report_generation.py:426-451`

在 `cluster_articles_for_report()` 中，`article_dicts` 构造完成后、调用 `_entity_report_async()` 之前，插入：

```python
from src.application.dedup import deduplicate_articles
# ...
deduped = deduplicate_articles(article_dicts)
return asyncio.run(_entity_report_async(deduped, ...))
```

注意：`deduplicate_articles` 需要 `id`、`content_hash`、`minhash_signature` 字段存在于 dict 中，这些在步骤1和2中已补充。
