# Quick Task 260409-r5y: 修复日报翻译 — 英文→中文

## Task

修复 `--language zh` 时英文标题不被翻译成中文的问题。

## Problem

`render_report` 中的预翻译逻辑 `if target_lang != "zh"` 导致 `--language zh` 时跳过所有标题翻译。英文标题保持原样。

## Solution

修改 `render_report` 中的预翻译条件，始终执行预翻译，但只翻译"需要翻译"的标题：
- `target_lang == "zh"` 且标题不是中文 → 翻译英文→中文
- `target_lang != "zh"` 且标题是中文 → 翻译中文→其他语言

## Files

- `src/application/report.py` — 修改 `render_report` 函数（约 5 行逻辑变更）

## Action

1. 找到 `render_report` 中 `if target_lang != "zh":` 条件块
2. 替换为始终执行的逻辑，但根据 target_lang 决定哪些标题需要翻译
3. 保持 `_is_chinese()` 用于判断标题语言

## Verify

```bash
uv run feedship --debug report --since 2026-04-07 --until 2026-04-10 --language zh
```

英文标题应有中文翻译（如 "The Illusion of..." → "The Illusion of...（中文翻译）"）
