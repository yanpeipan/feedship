## Key Data Structures

ClassifyTranslateOutput (from get_classify_translate_chain):
  items: [{id: 1-indexed int, tags: [str], translation: "translated title"}]

EntityTopic (expected by TLDRGenerator):
  entity_id: str
  entity_name: str
  layer: str (AI应用/AI模型/AI基础设施/芯片/能源)
  headline: str (summary headline)
  dimensions: dict[str, list[ArticleEnriched]]
  articles_count: int
  signals: list[str]
  tldr: str
  quality_weight: float

## Conversion Logic

1. Map ClassifyTranslateItem.id (1-indexed) back to original article at index (id-1)
2. Group articles by primary tag (tags[0]) or feed_id if no tags
3. For each group:
   - entity_id = primary_tag or feed_id
   - entity_name = primary_tag or feed_name
   - headline = translated title of highest-quality article in group
   - layer = classify_dimensions() on representative article
   - dimensions = group articles by classify_dimensions result
   - quality_weight = sum(quality_score) * count

   # Convert ClassifyTranslateOutput to EntityTopic[]
   # id is 1-indexed position in news_list, map back to original article
   # Group by primary tag (tags[0]) or feed_id if no tags
   _DIMENSION_KEYWORDS = {
       "release": ["release", "launch", "announce", "发布", "推出", "launches", "unveils"],
       "funding": ["funding", "raise", "series", "vc", "invest", "融资", "投资", "raises"],

       "policy": ["regulation", "policy", "government", "ban", "监管", "政策"],
   }
   def _classify_dim(article: dict) -> list[str]:
       text = ((article.get("title", "") or "") + " " + (article.get("summary", "") or "")).lower()
       dims = []
       for dim, kws in _DIMENSION_KEYWORDS.items():
           if any(kw in text for kw in kws):
               dims.append(dim)
       return dims if dims else ["ecosystem"]

   # Build index: item_id (1-indexed) -> original article dict
   articles_by_idx = {item.id: filtered[item.id - 1] for item in classify_output.items if item.id <= len(filtered)}

   # Group by primary tag (or feed_id as fallback)
   from collections import defaultdict
   tag_groups: dict[str, list[tuple[int, dict]]] = defaultdict(list)  # tag -> [(item_id, article_dict)]
   for item in classify_output.items:
       if item.id <= len(filtered):

           primary_tag = item.tags[0] if item.tags else art.get("feed_id", "unknown")
           tag_groups[primary_tag].append((item.id, art))

   # Also store translated title per item_id
   trans_by_id = {item.id: item.translation for item in classify_output.items}

   # Build EntityTopic for each tag group
   entity_topics: list = []
   from src.application.report.models import ArticleEnriched, EntityTopic, EntityTag
   for tag, items in tag_groups.items():

       # Build ArticleEnriched for each article
       article_enriched_list = []
       for art in arts:
           ae = ArticleEnriched(
               id=art.get("id", ""),
               title=art.get("title", ""),
               link=art.get("link", ""),
               summary=art.get("summary", ""),
               quality_score=art.get("quality_score", 0.0),

               entities=[],  # No entities available from classify_translate

           )
           article_enriched_list.append(ae)

       # Group by dimension
       by_dim: dict[str, list[ArticleEnriched]] = {}
       for ae in article_enriched_list:
           for dim in ae.dimensions:
               by_dim.setdefault(dim, []).append(ae)

       # Find best article by quality for headline
       best_art = max(arts, key=lambda a: a.get("quality_score", 0.0))
       best_ae = next(ae for ae in article_enriched_list if ae.id == best_art.get("id"))
       best_idx = next(i for i, a in enumerate(arts) if a.get("id") == best_art.get("id"))
       item_id = items[best_idx][0]
       headline = trans_by_id.get(item_id, tag)[:30]

       entity_topics.append(EntityTopic(
           entity_id=tag,
           entity_name=tag,
           layer="AI应用",
           headline=headline,
           dimensions=by_dim,
           articles_count=len(arts),
           signals=[],
           tldr="",
           quality_weight=sum(a.get("quality_score", 0.0) for a in arts) * len(arts),
       ))
